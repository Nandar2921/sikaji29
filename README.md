# SiKAJI - Sistem Informasi Kajian Islam

Platform kajian Islam modern dengan Al-Quran, Tafsir, dan Hadits.

## ✨ Fitur

- 📖 **Al-Quran Digital** - Baca Al-Quran dengan teks Arab dan terjemahan
- 📚 **Tafsir Lengkap** - Penjelasan mendalam dari berbagai ulama
- 🔍 **Smart Search** - Cari ayat atau surah dengan cepat
- 📜 **Hadits Nabawi** - Kumpulan hadits shahih
- 🌙 **Dark Mode** - Tampilan nyaman di malam hari
- 🔖 **Bookmark** - Simpan ayat favorit
- 🕐 **Riwayat Pencarian** - Lacak sejarah pencarian
- 📄 **Export PDF** - Download hasil pencarian
- 🔊 **Audio Murottal** - Dengarkan lantunan ayat

## 🛠️ Teknologi

- Next.js 14
- TypeScript
- Tailwind CSS
- PostgreSQL
- Drizzle ORM
- NextAuth.js
